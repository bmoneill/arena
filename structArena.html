Detailed Description

Arena structure.

This structure represents an arena of memory. It contains information about the arena's memory, pointer, head block, index, size, maximum block count, and management status.

Parameters
memA pointer to the memory block of the arena.
ptrA pointer to the current position in the memory block.
headA pointer to the head block of the arena.
idxThe index of the current block within the arena.
sizeThe size of the memory block in bytes.
maxBlocksThe maximum number of blocks that can be allocated in the arena.
managedA flag indicating whether the arena is managed or not.
